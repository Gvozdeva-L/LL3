# LL3

## Опис проєкту
Цей проєкт створено для ознайомлення із базовими діями роботи з Git та GitHub,
а також навчитися структурно організовувати репозиторій і ефективно працювати з гілками.

## Призначення репозиторію
Репозиторій призначений для навчання роботі з системою контролю версій Git:
створення, зміни, збереження та публікація файлів на GitHub.

## Опис структури
- index.html - html код
- src/ - JavaScript файл
- LOL.txt - порожній текстовий файл
- README.md - опис проєкту
